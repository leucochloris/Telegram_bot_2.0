# Telegram bot for agregaty customers data
## @school_easy_way_bot

Abstract client - an ordinary entrepreneur (owner of an English language school) which needs a channel to attract customers.

In the bot- there are several roles:

- client (select)
- administrator (SIUD)
- operator (select)
